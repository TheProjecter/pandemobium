Pandemobium Stock Trader - Vulnerabilities Guide

Pandemobium Stock Trader mobile applications and web services are designed to demonstrate common vulnerabilities.Below is a list of flaws that have been included

  • SQL Injection flaws are included and can be exploited on all input fields. Poor validation allows improper input to be treated as SQL statement.

  • Local storage file permissions are saved with world readable or normal permissions. World readable permissions allow other applications to access them. Normal permissions allows the files to be read offline such as when the phone is lost. A local database with world readable permissions is also included, allowing other users and applications to have access.

  • XSS flaws have been added by using incorrect input validation and output encoding. The web services do this by returning the echo invalid method names that are passed in.

  •   Poor trust relationships and boundaries have been added allowing invalid data to be submitted to the web services. The web services trusts the client to send valid stock prices.
